# AI Admission Chatbot

## Project Overview
This project is an AI-powered Admission Chatbot developed using IBM watsonx Assistant. It helps students by answering admission-related questions instantly.

## Features
- Student admission guidance
- Course information
- Fee details
- Eligibility information
- Contact details

## Technologies Used
- IBM watsonx Assistant
- IBM Cloud
- Twilio WhatsApp Integration
- GitHub

## How to Use
1. Open the chatbot.
2. Type your admission-related question.
3. Receive an instant response.
